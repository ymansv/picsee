## Picsee 到底是什么

Picsee 是 macOS + iOS 平台的图片管理软件，出自国内独立开发者 @chitaner 之手，功能覆盖图片采集、整理分类、查找查看、跨设备协同这一整套流程。

👉 [点击了解 Picsee 详情 / 购买专业版](https://lizhi.shop/products/picsee?cid=7bbix2xj)

说白了，就是把你电脑里乱成一锅粥的图片、截图、视频素材，用一个地方统一管起来。

跟 Eagle 比，Picsee 的逻辑稍微不一样——它不强制把文件拷贝进自己的库，而是直接关联你的本地文件夹，跟 Finder 实时双向同步。换句话说，你在 Finder 里移动了文件，Picsee 里马上就反映出来了，不用重新导入。

这个设计对于本来就有一套自己文件整理习惯的人来说相当友好。

## 用起来什么感觉

### 收集图片这块做得挺顺手

Picsee 提供了好几种收集方式：

- **浏览器插件**：网页上直接右键或者点一下，整页或者选中批量收藏，截图也可以直接扔进去
- **截图标注**：用快捷键 `Shift + Command + X` 触发截图，支持滚动截屏、马赛克、标注，截完直接存进 Picsee
- **拖放**：直接把文件拽进去，最省事
- **导入外部资源包**：支持 Eagle 素材包和花瓣面板直接导入，换软件的迁移成本几乎为零

之前用过几个截图工具，Picsee 截图功能虽然不是最强的，但直接把截图收进素材库这个流程做得很顺，省掉了截图完再手动整理这一步。

### 整理这块是真强项

标签、文件夹、星标、注释这些是基础，Picsee 的重点是**智能文件夹/任务**——可以根据颜色、尺寸、标签、名称等属性设定规则，让软件自动把素材归类。不用手动一张张拖，设置好规则就行了。

颜色筛选也是个很实用的功能——选一个主色调，Picsee 就把相近颜色的素材都找出来，对设计师来说找配色参考相当方便。

### 格式支持挺全

支持 PNG、JPG、HEIC、RAW（cr2/dng/nef 等）、WebP、SVG、AI、EPS、PSD，以及 MP4、MOV 等视频格式，还有 PDF、文档格式的预览。官方说超过 50 种格式，日常用遇到不支持的情况不多。

### 跨设备同步

通过 iCloud 同步标签、注释、星标等数据（不是文件本身，所以同步量很小），多台 Mac 之间协同还不错。iOS 端的 Picsee 可以免费使用，移动端看图配 Mac 管理，组合用起来还挺顺手。
